# 百度网盘协作技能包 — 使用说明

让 AI 智能体帮你操作百度网盘。上传、下载、搜索、分享，一句话搞定。

## 前置条件

1. 有百度网盘账号（没有的话免费注册一个）
2. 手机上有百度网盘 App

## 技能包内容

```
baidu-netdisk-workflow/      (~195MB)
├── SKILL.md                  ← AI 读取学习
├── 用户手册.md                ← 本文件
└── offline/                  ← 各平台离线安装包
    ├── mac_amd64/
    ├── mac_arm64/
    ├── linux_amd64/
    ├── linux_arm64/
    ├── win_amd64/
    └── win_arm64/
```

## 使用方式

把文件夹路径告诉你的 AI，让它读取学习。

**你跟 AI 说：**
> 我的技能包在 `~/Desktop/baidu-netdisk-workflow/`，里面有 SKILL.md，你先读取学习一下

**AI 学会后，一句话干活：**
> 帮我把桌面的报告.pdf 上传到网盘
> 帮我在网盘里搜一下关于合同的文件
> 帮我生成一个分享链接，7天有效

## 首次使用流程

1. AI 自动检测你的系统平台（macOS/Linux/Windows）
2. AI 尝试从 GitHub 下载工具，网络不通则用技能包自带的离线包
3. AI 引导你用手机百度网盘 App 扫码登录
4. 扫码成功后就可以用了

## 注意事项

- 登录 Token 有效期约 **30 天**，到期需要重新扫码
- 分享功能需要百度网盘付费接口支持
- 支持 macOS / Linux / Windows
- 技能包约 195MB（含各平台离线安装包）
